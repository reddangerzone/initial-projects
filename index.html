The Box Model
Tag Name Function
width: Width Establishes the width of the content area
height: Height Establishes the height of the content area
padding: Padding Establishes the amount of area between the content area and the border
border: Border Establishes the width and style of the border around the content area
margin: Margin Establishes the distance between the border and the edge of the page

Typography
Tag Name Function
@font-face{} Font Face Links to a custom font
font: Font Shorthand property that sets different properties for an element's font in a single declaration.
font-family: Font Family Specify the typeface in a rule set.
font-size: Font Size Sets the text size. Font size values can be in different units like absolute lengths and relative lengths.
font-style: Font Style To set the font style in which text will appear.
font-weight: Font Weight To set the text to be thicker or thinner.
letter-spacing: Letter Spacing Set the horizontal spacing between the individual characters in an element.
line-height: Line Height Set the vertical spacing between lines of text.
text-align: Text Align To set the text alignment of inline contents.
text-decoration: Text Decoration To add lines on the text.
text-indent: Text Indent Leaves empty space on the first line in a text-block.
text-justify: Text Justify Sets the justifcation method of text when text-align: justify; is applied to an element.
text-overflow: Text Overflow Specifies how hidden content is signaled to the user.
text-shadow: Text Shadow Adds shadow to text.
text-transform: Text Transform Specifies how to capitalize an elementâ€™s text.
word-spacing: Word Spacing Set space between words.
